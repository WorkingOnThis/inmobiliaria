# Product

## Register

product

## Users

Argentine real estate agency staff: agents and account admins who manage rental properties daily. They work at a desk, in a dim office or at home, running multiple operations simultaneously. Primary task on any screen is data entry, review, or action on a property, client, or contract. They are not tech-averse but they are not developers; they expect the tool to be fast and unambiguous.

## Product Purpose

Arce Administración is a full-cycle rental property management system for Argentine agencies. It handles tenants, owners, properties, contracts, cash movements, service tracking, document generation, and owner settlements. Success means the agent completes a task without friction and the data is trustworthy.

## Brand Personality

Professional, clear, trustworthy. Three words: precise, grounded, efficient. The interface should feel like a well-made tool, not a startup product. No decorative flourishes. Every element earns its place by reducing cognitive load or preventing errors.

## Anti-references

- Clunky legacy real estate desktop software (too dense, no hierarchy)
- Generic SaaS dashboards with big gradient numbers and empty charts
- Consumer apps that prioritize delight over clarity
- Anything that looks like a Tailwind UI template screenshot

## Design Principles

1. The task disappears into the interface. The agent should think about the contract, not the UI.
2. One primary action per screen. Secondary actions are secondary visually.
3. Errors are caught early, explained plainly, and recoverable.
4. Consistent vocabulary throughout. The same concept always has the same name.
5. Dark mode is not an aesthetic choice here — it is the environment. Design for it.

## Accessibility & Inclusion

WCAG AA minimum. Keyboard navigation required for all form flows. No color-only state communication.
