# Ledger Monto Manual — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Permitir editar manualmente el monto de cualquier ítem del `tenant_ledger` antes de cobrar, preservando el monto original para trazabilidad.

**Architecture:** Se agregan dos columnas a `tenant_ledger` (`montoOriginal`, `montoManual`). El endpoint existente `PATCH /api/tenants/[id]/ledger/[entryId]` recibe el nuevo campo. La UI del `LedgerTable` suma un lápiz inline por fila. La función `getEffectiveAmount` del emit usa `montoManual` como base si existe.

**Tech Stack:** Next.js App Router · Drizzle ORM · PostgreSQL · shadcn/ui · TanStack Query · Zod · lucide-react

function getEffectiveAmount(
