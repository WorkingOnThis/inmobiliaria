# Decisiones — Inquilinos

Módulo de gestión de inquilinos: ficha, cuenta corriente, recibos, acceso.

---

## Login de inquilinos y carga de comprobantes

**Estado:** `idea`

Permitir que los inquilinos se logueen en la app, vean su cuenta corriente y suban comprobantes de pago para que el admin los apruebe.

**Por qué posponerlo:** requiere definir roles de acceso diferenciados (inquilino vs. admin), UI separada, y flujo de aprobación. Es un módulo completo por sí solo.

**Cuándo revisarlo:** después de completar el MVP contable (recibos, liquidaciones, caja).
